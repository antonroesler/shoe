# shoe
A simple commandline app for keeping track of your running shoes mileage with as little overhead as possible.
# Installation
```bash
  $ git clone https://github.com/antonroesler/shoe
  $ cd shoe
  $ ./install
```
# Usage
```bash
$ shoe
```
## Create a new shoe
```bash
$ shoe create nike
```
## After a run
```bash
$ shoe add nike 11.7
```
